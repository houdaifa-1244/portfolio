Desktop Password Generator

This desktop application helps you generate strong passwords and store credentials securely. Add entries with a name (label), website, email and password — or auto-generate a secure password. Quickly find any saved password by searching with its name.

  • Add Entry: create a new credential with name, website, email and password (or generate one).
  • Search: find an entry by name and reveal the stored password safely.
  • Security tip: store data encrypted and protect the app with a master password for desktop use.

1. Add Entry

Fill the form: Name (label), Website, Email and Password. Use the generator button to create a strong password automatically (length, symbols, numbers options).

2. Find Password by Name

Search by the entry name (label) to quickly locate the credential. Select the result to reveal or copy the password. For safety, require the master password or biometric unlock before revealing.

Application Web : To-Do List

Accéder au site

Cette application To-Do List est une solution simple et moderne pour organiser vos tâches quotidiennes. Elle permet d’ajouter, compléter, supprimer et filtrer les tâches, avec un design clair/sombre et une interface fluide.


  • Ajout de tâche : permet d’ajouter rapidement une nouvelle tâche.
  • Compléter/Sélectionner : marque une tâche comme terminée.
  • Supprimer : retire la tâche de la liste.
  • Filtrage : affiche toutes les tâches, actives ou terminées.
  • Mode sombre : change automatiquement le thème de l’application.

1. Interface principale

L’interface principale affiche la liste des tâches avec un design moderne et intuitif.

2. Ajouter une tâche

L’utilisateur peut ajouter une nouvelle tâche à l’aide du champ de saisie et du bouton “Add”.

3. Sélectionner une tâche terminée

En cliquant sur une tâche, elle est marquée comme terminée (grisée ou barrée).

4. Filtres des tâches

Trois filtres sont disponibles : All, Active, et Completed. Un bouton Clear Completed permet de supprimer toutes les tâches terminées.

5. Mode sombre

L’application inclut un Dark Mode élégant qui s’active automatiquement selon le thème choisi.

Reservation Terrain App

Visit the App

Welcome to Reservation Terrain

A complete reservation platform for sports terrains. Users can sign in or create an account, add and manage reservations (choose sport, terrain, date and time) while admins manage users, sports, terrains, closing hours and approve or cancel reservations.

1 — Login

Sign in with email and password to access your account and dashboard.

2 — Create Account

Register with name, email and password to create a new user account.

3 — User Dashboard

Access your dashboard to add reservations and view your list of bookings.

4 — Add Reservation

Fill the reservation form: name, sport, terrain, date, start time and end time.

5 — Reservations List

View and manage all your existing reservations: edit or cancel when needed.

6 — Admin Dashboard

Central admin area: add/manage users, sports, terrains, closing hours and view reservations.

7 — Manage Admins

Create, edit or remove admin accounts and assign roles/permissions.

8 — Manage Sports

Manage the list of sports (add, edit, deactivate categories used by reservations).

9 — Add Sport

Add new sport categories with name, icon and description for users to choose.

10 — Add Terrain

Add new terrains/fields (name, location, sport type, capacity, images).

11 — Manage Terrains

Edit terrain details, enable/disable availability and attach images.

12 — Add Closing Hours

Define daily closing hours or unavailable time slots for each terrain.

13 — List Reservations (Admin)

Full reservations list with filters (date, terrain, user, status).

14 — Accept / Cancel Reservation

Approve or cancel reservation requests and notify users instantly.
